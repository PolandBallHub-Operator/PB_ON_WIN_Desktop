# PolandBall Desktop App

このアプリは、デスクトップ上にポーランドボールを表示し、物理演算によって転がしたり、マウスでドラッグして投げたりできるWindows用ネイティブアプリケーションです。

## 内容物
- `PolandBall.exe`: 実行ファイル
- `main.cpp`: ソースコード（C++/Win32 API + GDI+）
- `balls/`: アプリで使用する画像リソース

## 操作方法
- **左クリックドラッグ**: ボールを掴んで動かしたり、投げたりできます。
- **右クリック**: アプリを終了します。

## 技術詳細
- **言語**: C++
- **GUI**: Win32 API (Layered Window)
- **描画**: GDI+
- **コンパイラ**: MinGW-w64 (x86_64)
